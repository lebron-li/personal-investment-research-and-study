# A 股财务报表深度分析器

对 A 股上市公司进行全面财务分析，评估盈利质量、财务健康和造假风险。

## 工作流程

### 第一步：确定对象
- 公司：股票代码/名称
- 时间跨度：默认 5 年（20 季度）
- 对比公司：主要竞争对手

### 第二步：杜邦分析（5 因子）

```
ROE = 税负系数 × 利息负担 × 营业利润率 × 资产周转率 × 权益乘数
```

### 第三步：盈利质量测试

| 测试 | 红灯阈值 |
|------|---------|
| 应计比率 | > 总资产 10% |
| 现金转化率 | 持续 < 0.8 |
| 应收增速 | 超过收入增速 |
| 政府补助依赖 | > 净利润 30% |

### 第四步：财务健康评分

| 模型 | 用途 |
|------|------|
| Altman Z 值 | 破产预测 |
| Piotroski F 值 | 财务实力（9 个信号） |
| Beneish M 值 | 盈利操纵检测 |

### 第五步：A 股特有风险检查

- 关联交易公允性
- 在建工程长期不转固
- 商誉减值风险
- 其他应收款（资金占用）
- 大股东股权质押

## 数据获取方式

**巨潮资讯**：http://www.cninfo.com.cn/（法定披露）
**东方财富**：https://data.eastmoney.com/bbsj/

## 重要提醒

- 关注扣非净利润
- 读附注（最重要信息在附注中）
- A 股特有陷阱：关联交易、政府补助、商誉减值
- 趋势比绝对值更重要
- 以上分析仅供参考，不构成投资建议
