# A 股量化因子筛选器

使用多因子模型筛选 A 股，包含中国特色因子。

## 因子体系

### 价值因子
- PE、PB、PCF、PS
- EV/EBITDA
- 股息率

### 成长因子
- 营收增速
- 利润增速
- ROE 变化

### 质量因子
- ROE、ROIC
- 毛利率
- 应计比率

### 动量因子
- 12 月动量（剔除最近 1 月）
- 相对强度

### 中国特色因子

| 因子 | 逻辑 |
|------|------|
| 换手率 | 低换手率因子在 A 股显著有效 |
| 北向资金 | 外资持续买入有超额收益 |
| 分析师覆盖 | 覆盖度提升预示关注度上升 |
| 国企/民企 | 国企估值修复逻辑 |

## 因子择时

基于宏观数据调整因子权重：

| 宏观环境 | 超配因子 |
|---------|---------|
| PMI 扩张 | 成长、动量 |
| PMI 收缩 | 价值、低波 |
| 社融扩张 | 小盘、成长 |
| 社融收缩 | 大盘、价值 |

## 工作流程

1. 确定股票池（沪深 300/中证 500/全 A）
2. 计算各因子得分
3. 宏观环境判断
4. 因子加权综合得分
5. 筛选前 N 名

## 重要提醒

- 因子有效性会衰减
- 避免过度拟合
- 考虑交易成本
- 以上分析仅供参考，不构成投资建议
