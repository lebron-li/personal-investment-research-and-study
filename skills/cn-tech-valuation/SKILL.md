# A 股科技股估值分析器

区分科技公司的概念炒作与基本面支撑，评估合理估值区间。

## 核心分析框架

### 第一步：技术/产品真实性评估

| 维度 | 验证方法 |
|------|---------|
| 技术壁垒 | 专利数量、研发投入、技术人员占比 |
| 产品落地 | 收入贡献、客户名单、订单情况 |
| 行业地位 | 市场份额、竞争对手对比 |
| 概念含"AI"量 | AI 相关收入占比 |

### 第二步：估值方法选择

| 公司类型 | 适用方法 |
|---------|---------|
| 成熟盈利 | PE、PEG、EV/EBITDA |
| 高增长未盈利 | PS、EV/Sales、用户价值 |
| 平台型 | GMV/用户数×变现率 |
| 硬科技 | 研发投入资本化调整 |

### 第三步：泡沫检测

| 信号 | 警示 |
|------|------|
| PE > 100 且增速 < 30% | 高估 |
| PS > 20 且无盈利路径 | 高风险 |
| 研发费用率 < 5% 但宣称高科技 | 概念炒作 |
| 大股东频繁减持 | 内部人不看好 |

### 第四步：合理估值区间

基于 PEG、同行对比、历史分位给出合理估值区间。

## 数据获取方式

**东方财富**：https://data.eastmoney.com/
**巨潮资讯**：http://www.cninfo.com.cn/
**专利局**：http://www.cnipa.gov.cn/

## 重要提醒

- 科技股估值需要区分真成长 vs 概念
- 高估值可以持续很久，但也可能快速崩塌
- 关注大股东和管理层行为
- 技术迭代风险高
- 以上分析仅供参考，不构成投资建议
