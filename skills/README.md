# 金融分析技能集

本目录包含 20 个金融分析技能，按功能分类如下。

---

## 估值分析

| 技能 | 说明 |
|------|------|
| 估值模型方法论 | DCF、DDM、SOTP 等绝对估值；PE-Band、PB-ROE、EV-EBITDA 等相对估值；动态 Ke 调整；利差安全边际；估值陷阱识别；叙事-数字交叉验证 |
| cn-tech-valuation | A 股科技股估值分析器。区分科技公司的概念炒作与基本面支撑，评估合理估值区间 |
| cn-undervalued-screener | A 股低估值股票筛选器。使用 PE/PB/ROE/现金流等多维度指标筛选被低估的上市公司 |
| cn-financial-analyzer | A 股财务报表深度分析器。杜邦分析、盈利质量评估、Z/M/F 值财务健康评分、法证财务分析 |

## 组合管理

| 技能 | 说明 |
|------|------|
| cn-portfolio-health | A 股组合健康诊断工具。集中度、相关性、流动性风险、A 股特有压力测试 |
| cn-risk-optimizer | A 股风险调整收益优化器。夏普比率、最大回撤等指标构建最优组合 |
| cn-long-term-builder | A 股长期建仓助手。以年为单位回报周期，日线/周线技术指标寻找低价收集优质筹码的入场时机 |

## 策略与量化

| 技能 | 说明 |
|------|------|
| cn-quant-screener | A 股量化因子筛选器。多因子模型，含换手率、北向资金等中国特色因子，PMI/社融因子择时 |
| cn-high-dividend | A 股高股息策略分析器。评估分红可持续性与长期回报，覆盖中证红利指数成分股 |
| cn-sector-rotation | A 股行业轮动信号探测器。基于美林时钟框架，通过宏观经济指标识别行业轮动机会 |

## 事件驱动

| 技能 | 说明 |
|------|------|
| cn-event-driven | A 股事件驱动机会识别器。资产重组、国企改革、分拆上市、回购增持等 |
| cn-insider-trading | A 股董监高增减持分析器。识别管理层信心信号 |
| cn-sentiment-gap | A 股市场情绪与基本面偏差分析。识别被过度看空但基本面稳健的逆向投资机会 |
| cn-small-cap-growth | A 股小盘成长股发现器。发现被市场忽视的小市值高成长公司（20-200 亿元） |

## 技术分析

| 技能 | 说明 |
|------|------|
| cn-tech-analysis | A 股技术面分析报告生成器。指标解释、形态识别、分周期操作建议 |
| cn-daily-report | A 股持仓日报自动生成器。每日分析持仓技术面、资金面、共振信号 |
| cn-midday-briefing | A 股午间快报。每个交易日 11:35 结合日K技术面+上午分时走势输出操作提示 |

## 宏观与 ESG

| 技能 | 说明 |
|------|------|
| macro-liquidity | 宏观流动性监控与风险预警。追踪美联储净流动性、SOFR、MOVE 指数、日元套利交易信号 |
| cn-esg-screener | A 股 ESG 筛选器。环境、社会、治理视角评估，覆盖双碳目标、共同富裕等中国特色 ESG 框架 |
| cn-suitability-report | A 股投资适当性报告生成器。符合证监会适当性管理框架 |
