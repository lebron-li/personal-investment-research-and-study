# A 股组合健康诊断

诊断现有 A 股组合的风险状况。

## 诊断维度

### 1. 集中度风险

| 指标 | 健康标准 |
|------|---------|
| 单一持仓占比 | < 20% |
| 前 5 大持仓 | < 60% |
| 单一行业占比 | < 40% |
| 白酒/新能源聚集度 | 警惕过度集中 |

### 2. 相关性分析

| 风险 | 说明 |
|------|------|
| 产业链共振 | 持仓处于同一产业链（如新能源上中下游） |
| 风格暴露 | 全部是大盘/小盘、价值/成长 |
| 地域集中 | 全部是某省份公司 |

### 3. 流动性风险

| 指标 | 警示 |
|------|------|
| 日均成交额 | < 5000 万 |
| 涨跌停频率 | 频繁跌停 |
| 质押比例 | 大股东高比例质押 |

### 4. A 股特有风险

- 政策风险（监管、产业政策）
- 解禁压力（限售股到期）
- 退市风险（ST/*ST）
- 财务造假风险

### 5. 压力测试

模拟以下情景的组合回撤：
- 大盘下跌 20%
- 行业政策利空
- 流动性危机

## 输出格式

```markdown
# 📋 组合健康诊断报告

## 综合评分：[X/100]

## 风险雷达
[各维度评分]

## 主要问题
[识别的风险点]

## 改进建议
[具体调仓建议]
```

## 数据获取方式

**东方财富**：https://data.eastmoney.com/
**同花顺**：http://data.10jqka.com.cn/

## 重要提醒

- 定期诊断（建议每季度）
- 分散不是目的，风险调整后收益才是
- 以上分析仅供参考，不构成投资建议
