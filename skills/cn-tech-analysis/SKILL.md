# cn-tech-analysis - A 股技术面分析报告生成器 (新手友好版)

## 技能描述

为用户的持仓股票和 ETF 生成**新手友好的**技术面分析报告，包含：
- ✅ 技术指标解释（新手必读）
- ✅ 技术形态识别（金叉/死叉/共振等）
- ✅ 基本面数据（PE、PB、行业）
- ✅ 分周期操作建议（短期/中期/长期）
- ✅ 置信度评估

**支持范围**:
- ✅ A 股股票 (600xxx, 000xxx, 300xxx)
- ✅ ETF 基金 (515xxx, 159xxx 等)

## 核心功能

### 1. 自动读取持仓列表
从 `C:\Users\李正材\Desktop\The stocks and ETFs I bought.txt` 读取股票代码和 ETF 代码

### 2. 技术指标分析
| 指标 | 说明 | 新手解读 |
|------|------|----------|
| MA | 移动平均线 | 站上为强，跌破为弱 |
| MACD | 趋势指标 | 金叉看涨，死叉看跌 |
| RSI | 超买超卖 | <30 超卖，>70 超买 |
| KD | 买卖点 | 金叉买点，死叉卖点 |
| 布林带 | 价格通道 | 上轨压力，下轨支撑 |
| ADX | 趋势强度 | <25 震荡，>25 趋势 |

### 3. 基本面数据
- **市盈率 (PE)**: 衡量估值，<15 低估值，>50 高估值
- **市净率 (PB)**: 股价/净资产，<1 破净，1-3 合理，>3 偏高
- **所属行业**: 股票所属行业分类

### 4. 技术形态识别
自动识别以下技术场景：
- 🟢 **均线多头排列**: 上升趋势确立，可持股待涨
- 🔴 **均线空头排列**: 下降趋势确立，谨慎操作
- 🟢 **MACD 金叉**: 多头动能增强，可考虑买入
- 🔴 **MACD 死叉**: 空头动能增强，考虑减仓
- 🟢 **RSI 超卖**: 短期跌幅过大，可能反弹
- 🔴 **RSI 超买**: 短期涨幅过大，可能回调
- 🟢 **KD 金叉**: 短期买点信号
- 🔴 **KD 死叉**: 短期卖点信号
- 🟡 **震荡市**: 箱体波动，高抛低吸
- 🟡 **趋势市**: 趋势明确，顺势操作
- 🟢 **多头共振**: 多个看涨信号，强烈建议持有
- 🔴 **空头共振**: 多个看跌信号，强烈建议减仓

### 5. 分周期操作建议
| 周期 | 持有期 | 策略特点 |
|------|--------|----------|
| 短期 | <5 天 | 严格止损，可追涨 |
| 中期 | 5-20 天 | 等待确认，控制仓位 |
| 长期 | >20 天 | 趋势跟踪，分批建仓 |

### 6. 持仓总结
- 技术面排名（从高到低）
- 可加仓标的
- 持有观望标的
- 注意风险标的
- 整体仓位建议

## 使用方法

### 方式 1：直接请求分析报告
```
"给我一份技术面分析报告"
"分析一下我的持仓"
```
→ 自动分析持仓列表中所有股票和 ETF

### 方式 2：指定股票代码
```
"分析一下 600036 的技术面"
"招商银行最近走势如何"
```
→ 分析指定股票

### 方式 3：指定 ETF 代码
```
"515170 现在能加仓吗"
"创新药 ETF 技术面分析"
```
→ 分析指定 ETF

### 方式 4：命令行运行
```powershell
cd C:\Users\李正材\.openclaw\workspace\skills\cn-tech-analysis\scripts
C:\Users\李正材\Downloads\skills-main\stock-analysis\venv\Scripts\python.exe analyze_portfolio.py
```

## 注意事项

1. **数据来源**: 
   - 股票：akshare (新浪财经)
   - ETF：akshare (东方财富)

2. **分析频率**: 建议每日收盘后分析一次

3. **置信度**: 是概率评估，非绝对保证

4. **风险提示**: 
   - 分析结果仅供参考，不构成投资建议
   - 震荡市 (ADX<25) 时建议降低仓位
   - RSI 超买/超卖时警惕反转
   - 高估值股票 (PE>50) 注意回调风险

5. **新手建议**:
   - 优先选择评分≥70 的标的
   - 评分<55 的标的谨慎参与
   - 分批建仓，不要一次性重仓
   - 设置止损位，控制风险
