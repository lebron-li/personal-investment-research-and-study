# A 股风险调整收益优化器

构建风险调整后收益最优的 A 股投资组合。

## 优化目标

### 风险调整指标

| 指标 | 公式 | 目标 |
|------|------|------|
| 夏普比率 | (组合收益 - 无风险利率) / 波动率 | 最大化 |
| 索提诺比率 | 超额收益 / 下行波动率 | 最大化 |
| 卡玛比率 | 年化收益 / 最大回撤 | 最大化 |
| 信息比率 | 超额收益 / 跟踪误差 | 最大化 |

### 约束条件

| 约束 | 默认值 |
|------|-------|
| 单一持仓上限 | 20% |
| 单一行业上限 | 40% |
| 现金比例 | 5-10% |
| 换手率限制 | 年化<200% |

## 工作流程

### 第一步：输入组合

- 现有持仓及权重
- 或候选股票列表

### 第二步：风险测算

| 指标 | 计算 |
|------|------|
| 波动率 | 历史收益率标准差 |
| 相关性 | 股票间相关系数矩阵 |
| VaR | 在险价值（95%/99%） |
| 最大回撤 | 历史最大峰值到谷底 |

### 第三步：优化求解

使用均值 - 方差模型或其他优化算法求解最优权重。

### 第四步：敏感性分析

测试不同市场情景下的组合表现。

## 输出格式

```markdown
# 📊 组合优化报告

## 当前组合
[持仓及权重]

## 优化建议
[建议权重调整]

## 风险指标对比
[优化前后对比]

## 敏感性分析
[不同情景测试]
```

## 重要提醒

- 历史数据不代表未来
- 优化结果需结合主观判断
- 定期再平衡
- 考虑交易成本
- 以上分析仅供参考，不构成投资建议
