# A 股市场情绪与基本面偏差分析

识别被市场过度看空但基本面稳健的逆向投资机会。

## 工作流程

### 第一步：情绪指标采集

| 指标 | 来源 |
|------|------|
| 分析师评级变化 | 上调/下调比例 |
| 媒体情绪 | 新闻正面/负面比例 |
| 散户情绪 | 股吧/雪球讨论热度 |
| 资金流向 | 北向资金、主力资金 |
| 估值分位 | 历史 PE/PB 百分位 |

### 第二步：基本面指标采集

| 维度 | 指标 |
|------|------|
| 盈利 | 营收增速、利润增速、ROE |
| 现金流 | 经营现金流、FCF |
| 负债 | 资产负债率、利息覆盖 |
| 运营 | 存货周转、应收周转 |

### 第三步：偏差计算

```
偏差分数 = 情绪评分（反向） - 基本面评分
```

高偏差分数 = 情绪极度悲观但基本面稳健 = 潜在逆向机会

### 第四步：深度验证

对高偏差公司验证：
- 情绪低迷的真实原因
- 基本面是否真的稳健（有无隐藏风险）
- 催化剂：什么能修复情绪

## 数据获取方式

**东方财富**：https://data.eastmoney.com/
**雪球**：https://xueqiu.com/
**同花顺**：http://data.10jqka.com.cn/

## 重要提醒

- 情绪可以持续非理性，逆向投资需要耐心
- 确认基本面真的稳健（非价值陷阱）
- 等待催化剂出现
- 分批建仓，控制仓位
