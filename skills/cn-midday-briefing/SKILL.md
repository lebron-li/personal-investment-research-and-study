# cn-midday-briefing - A 股午间快报

## 技能描述

每个 A 股交易日 11:35（上午收盘后），自动为持仓股票和 ETF 生成**午间轻量快报**，包含：
- ✅ 上午走势复盘（开盘、午间价、振幅、量能）
- ✅ 技术面简要快照（评分/信号摘要，复用 cn-tech-analysis 指标计算）
- ✅ 关键位置提醒（价格与均线支撑/压力的距离）
- ✅ 下午操作提示（基于上午走势+技术面的综合建议）

**定位**：盘中午间快报，偏实战决策。**不做深度技术分析**（深度分析请用 cn-tech-analysis）。

## 与原版 cn-tech-analysis 的区别

| 维度 | cn-midday-briefing (本技能) | cn-tech-analysis |
|------|---------------------------|-------------------|
| 触发时机 | 交易日 11:35 | 收盘后/按需 |
| 数据范围 | 日K线 + 当天上午分时 | 纯日K线 |
| 分析深度 | 轻量快报，偏实战决策 | 深度分析，指标/形态/分周期 |
| 报告长度 | 每只标的 10-15 行 | 每只标的 30+ 行 |
| 核心输出 | 上午复盘 + 技术快照 + 下午提示 | 完整技术面分析报告 |
| 输出渠道 | 飞书群聊 / 终端 | 终端 / 本地文件 |

## 核心功能

### 1. 上午走势复盘
针对每只持仓标的，输出：
- 前日收盘价 → 今日开盘价 → 午间收盘价
- 上午最高价、最低价、上午振幅
- 上午涨跌幅（相对前日收盘）
- 开盘方向（高开/低开/平开）及幅度
- 上午成交量 vs 近期上午成交量均值（判断资金情绪）

### 2. 技术面简要快照
复用 cn-tech-analysis 的指标计算模块，输出：
- 综合评分（/100）
- 核心信号摘要（最多 5 条，精简版）
- 当前价格与关键均线（MA5/MA20/MA60）的位置关系

### 3. 关键位置提醒
- 最近支撑位（下方最近均线或前低）
- 最近压力位（上方最近均线或前高）
- 距离支撑/压力的百分比
- 下午重点关注价位

### 4. 下午操作提示
基于上午走势 + 技术面的综合判断：
- 各标的下午操作建议（可操作/观望/谨慎）
- 整体仓位建议
- 特别关注标的（异动提醒）

## 依赖项

- Python 3.10+
- pandas, numpy
- akshare 或直接 HTTP（东方财富/新浪）
- 复用 cn-tech-analysis 的指标计算逻辑

## 数据获取策略

### 日K线数据
- 获取最近 90 天日K（同上版）
- 来源：东方财富 HTTP API（首选，不依赖 akshare）

### 上午分时数据
- 获取当日 1分钟/5分钟 K 线，截取 9:30-11:30 区间
- 来源：东方财富 API（`klt=1` 或 `klt=5`）
- 兜底：新浪财经分时接口

### 午间数据可用性检查
- 若当日数据尚未生成（API 返回数据截止到昨日），视为**非交易日**，跳过分析
- 若分时数据不完整（不足 120 分钟），降级为仅日K分析并标注

## 注意事项

1. **交易日判断**：脚本会检测今日是否有交易数据，非交易日自动跳过
2. **上午数据延迟**：11:30 收盘后数据需 3-5 分钟同步，建议 11:35 触发
3. **量能对比**：上午量能与近期上午均值的对比仅为参考，不构成确定性判断
4. **风险警示**：午间快报仅提供参考信息，下午走势受多种因素影响，不构成操作指令
