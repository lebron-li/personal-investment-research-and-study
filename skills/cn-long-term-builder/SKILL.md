# cn-long-term-builder — A 股长期建仓助手

## 技能描述

专为**长期主义投资者**设计。核心理念：

> 先排除"因为基本面恶化所以便宜"的价值陷阱，再寻找"被市场错误定价"的收集机会——价格低≠值得买。

与 `cn-tech-analysis`（短中线交易导向）的本质区别：

| 维度 | cn-tech-analysis | cn-long-term-builder |
|------|-----------------|---------------------|
| 投资周期 | 5-20 天 | **1-5 年** |
| 核心问题 | 明天该不该买卖？ | **现在是不是收集筹码的好区间？** |
| 评分逻辑 | 趋势向上 = 高分 | **估值低+跌够了 = 高分** |
| 第一道防线 | 无 | **价值陷阱排雷（v2.0 新增）** |
| 技术指标角色 | 决策因子 | **入场时机校准** |
| 建议输出 | 买入/持有/减仓 | **可收集区间 / 等待区间 / 建议分批量** |

---

## 核心功能

### 0. 价值陷阱排雷（v2.0 新增 — Quality Gate）

在估值温度计读数之后、评分计算之前，插入质量过滤器。防止"低PE/PB是因为基本面恶化"的情况被误判为投资机会。

**银行股专用质量检测**（自动获取东方财富F10数据）：

| 维度 | 检测指标 | 红灯阈值 | 质量系数影响 |
|------|---------|---------|------------|
| ROE趋势 | 近3年ROE年化下滑幅度 | >2pp/年 → 🔴 / >1pp/年 → 🟡 | ×0.65 / ×0.80 |
| 净息差趋势 | 近8季NIM累计压缩(bp) | >15bp(低NIM) 或 >20bp(高NIM) → 🔴 | ×0.65 / ×0.80 |
| 拨备安全性 | 拨备覆盖率 | <200% → 🔴 / <250% → 🟡 | ×0.75 / ×0.85 |
| 不良认定 | 逾期/不良贷款比例 | >150% → 🔴 / >130% → 🟡 | ×0.75 / ×0.85 |
| 资本充足率 | 核心一级资本充足率(CET1) | <8% → 🔴 / <9% → 🟡 | ×0.80 / ×0.85 |

**非银行股通用质量检测**：

| 维度 | 检测指标 | 红灯阈值 | 质量系数影响 |
|------|---------|---------|------------|
| 盈利质量 | 经营现金流/净利润 | <0.5 → 🔴 / <0.7 → 🟡 | ×0.60 / ×0.80 |
| 分红可靠性 | 近3年是否存在中断/下降 | 中断 → 🔴 / 下降>10% → 🟡 | ×0.65 / ×0.85 |
| 商誉风险 | 商誉/净资产比例 | >30% → 🔴 / >20% → 🟡 | ×0.70 / ×0.85 |

质量系数（0-1）直接乘在估值温度计原始得分上。

### 1. 估值温度计 — 判断"够不够便宜"

| 指标 | 说明 | 长期建仓判断 |
|------|------|-------------|
| PE 分位 (5年) | 当前市盈率在历史中的排名 | < 20% 分位 → 极度低估，优先收集 |
| PB 分位 (5年) | 当前市净率在历史中的排名 | < 10% 分位 → 破净或接近破净 |
| 股息率 vs 国债 | 股息率与10年期国债收益率之差 | 差 > 1.5% → 防御性标的的黄金收集区 |

### 2. 周期位置 — 判断"跌够了没有"

| 指标 | 日线 | 周线 | 长期建仓判断 |
|------|------|------|-------------|
| 距52周高点回撤 | ✅ | ✅ | > 30% → 深度回调区 |
| 距52周低点距离 | ✅ | ✅ | < 10% → 接近底部 |
| ADX 趋势强度 | ✅ | ✅ | 周线 ADX > 25 + DI-占优 → 仍在下跌趋势中，分批 |

### 3. 技术面入场校准 — 日线+周线双周期

**日线**用于判断短期是否超卖/企稳：
- RSI < 30 → 短期超卖，可能反弹
- KD < 20 → 短期极端超卖
- 布林带下轨 → 价格处于统计低位
- 成交量萎缩后放量 → 底部换手信号

**周线**用于判断中期趋势：
- 周线 MACD 是否底背离 → 下跌动能衰减
- 周线 MA20/MA60 → 中期均线压制还是支撑
- 周线成交量趋势 → 大资金是否在暗中吸筹

### 4. 资金面 — 聪明钱在做什么

- 北向资金月度净买卖
- 融资余额趋势
- 股东户数变化（减少 = 筹码集中 = 利好长线）
- 大宗交易折溢价

### 5. 行业/周期因子

- 周期股：结合行业周期位置（如猪价、煤价、运价）
- 成长股：PEG、研发投入/营收比
- 红利股：分红稳定性、自由现金流/分红比
- ETF：跟踪指数的估值分位 + 底层资产集中度

## 评分体系（长线导向，与 cn-tech-analysis 完全反逻辑）

| 维度 | 权重 | 评分规则 |
|------|------|---------|
| **质量评分** 🆕 | **20%** | ROE/NIM趋势+拨备安全+不良认定 → 质量越高分越高 |
| 估值温度计 | 25% | PE分位越低越高分（**注意：已被质量系数调整**） |
| 周期位置 | 20% | 回撤越深越高分（跌出来的机会） |
| 技术面入场 | 15% | 日线超卖+周线底背离=高分 |
| 资金面 | 10% | 北向流入+筹码集中=高分 |
| 行业周期 | 10% | 行业底部区域=高分 |

**质量系数调整机制**：估值温度计原始得分 × 质量系数 = 调整后估值分。
- 例：某银行 PE分位极低 → 估值原始分 28/30，但 ROE加速下滑+NIM暴跌 → 质量系数 0.75
- 调整后估值分 = 28 × 0.75 = 21，而非 28

**评级阈值**：
- ≥ 80：🟢 黄金收集区 — 可积极建仓
- 70-79：🟢 可收集区间 — 分批建仓
- 55-69：🟡 观察等待区 — 轻仓持有，等待更好价格
- 40-54：🟡 暂不收集 — 等待估值或技术面改善
- < 40：🔴 远离区间 — 估值过高或趋势破位

## 注意事项

1. 这是一套**长期建仓辅助工具**，不是短期交易信号
2. 评分越高意味着当前价格越接近历史低估区间，不代表不会继续下跌
3. 分批建仓是核心策略——永远不要一次性打完子弹
4. 行业周期判断依赖外部数据（如猪价、运价），非纯技术面
