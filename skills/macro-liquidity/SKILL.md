# 宏观流动性监控系统

这个技能帮助你追踪全球金融体系中最关键的"水位"——流动性。流动性可以简单理解为市场里有多少钱在流动。钱多（流动性充裕）时，各类资产倾向于上涨；钱少（流动性收紧）时，资产价格承压。

## 使用场景

当用户询问以下类型问题时使用此技能：
- 现在流动性怎么样 / 美联储在放水还是缩表
- SOFR 利率走势如何
- 美债市场波动大不大
- 日元套利交易会不会爆
- 宏观环境对风险资产（股票、加密货币）友好吗

## 分析框架

### 4 大核心监控指标

#### 指标 1：美联储净流动性（Net Liquidity）

**是什么**：衡量美联储实际向市场注入了多少钱的核心公式：

**净流动性 = 美联储总资产 - TGA 余额 - ON RRP 余额**

- **美联储总资产**：美联储持有的国债和 MBS 总额。"印钱"买债券 → 注资；"缩表" → 抽水。
- **TGA（财政部一般账户）**：美国财政部在美联储的"银行账户"。TGA 增加 = 抽水；TGA 减少 = 放水。
- **ON RRP（隔夜逆回购）**：货币市场基金把多余现金"存"在美联储的工具。ON RRP 高 = 钱被"冻住"；ON RRP 低 = 钱回流市场。

**搜索关键词**：`Federal Reserve balance sheet total assets` + `Treasury TGA balance` + `ON RRP balance`

**预警标准**：
- 净流动性单周下降 > 5% → 🔴 预警：流动性急剧收缩
- 净流动性缓慢下降但趋势明确 → 🟡 关注：逐步收紧中
- 净流动性稳定或上升 → ✅ 正常：流动性环境友好

**解读要点**：净流动性与标普 500、比特币等风险资产的走势有很强的相关性。

---

#### 指标 2：SOFR（担保隔夜融资利率）

**是什么**：华尔街机构之间"借钱过夜"的利率。SOFR 异常升高意味着市场上的短期资金供不应求，机构们在"抢钱"。

**搜索关键词**：`SOFR rate today` 或 `secured overnight financing rate latest`

**预警标准**：
- SOFR > 美联储联邦基金利率上限 + 10bp → 🔴 预警：资金面极度紧张
- SOFR 接近联邦基金利率上限 → 🟡 关注：资金面偏紧
- SOFR 在联邦基金利率区间内正常波动 → ✅ 正常

**补充说明**：bp（基点）= 0.01%。10bp = 0.1%。

**解读要点**：SOFR 的突然飙升往往是流动性危机的先兆（如 2019 年 9 月"回购危机"）。

---

#### 指标 3：MOVE 指数（美债波动率指数）

**是什么**：衡量美国国债市场波动率的指标，类似于股市的 VIX（恐慌指数），但针对的是债券市场。

**搜索关键词**：`MOVE index today` 或 `MOVE bond volatility index`

**预警标准**：
- MOVE > 130 → 🔴 预警：债市剧烈波动，风险资产应考虑止损或对冲
- MOVE 100-130 → 🟡 关注：波动偏高，需密切监控
- MOVE < 100 → ✅ 正常：债市相对平静

**历史参考**：
- 2008 年金融危机：MOVE 一度超过 260
- 2020 年 3 月新冠恐慌：MOVE 飙升至 160+
- 2023 年 3 月硅谷银行危机：MOVE 达到 180+
- 正常市场环境：MOVE 通常在 60-100 之间

---

#### 指标 4：日元套利交易信号（Yen Carry Trade Monitor）

**是什么**：日元套利交易是全球最大的"借低利率货币、投高收益资产"策略。

**监控变量**：
- **USDJPY（美元兑日元汇率）**：汇率越高 = 日元越弱 = 套利交易越活跃
- **US2Y - JP2Y 利差**：美日 2 年期国债利差，这是套利交易的"利润来源"

**搜索关键词**：`USDJPY today` + `US 2 year treasury yield` + `Japan 2 year bond yield`

**预警标准**：
- USDJPY 单周下跌 > 3%（日元急剧走强）→ 🔴 预警：套利平仓风险
- 美日 2 年利差快速收窄（单月 > 50bp）→ 🔴 预警：套利基础在瓦解
- USDJPY 缓慢走弱 + 利差小幅收窄 → 🟡 关注
- USDJPY 稳定或走强 + 利差维持 → ✅ 正常

**历史案例**：2024 年 7-8 月日本央行意外加息 → 日元急升 → 全球套利交易大规模平仓 → 日经 225 单日暴跌 12%，美股、加密货币同步大跌。

---

### 综合评估逻辑

| 预警情况 | 流动性评级 | 应对建议 |
|---------|-----------|---------|
| 0 个预警 | 🟢 充裕 | 流动性环境友好，适合持有风险资产 |
| 1 个预警 | 🟡 偏紧 | 保持警惕，检查止损位，适度降低杠杆 |
| 2 个预警 | 🟠 紧张 | 减少风险敞口 10-20%，提高现金比例 |
| 3 个预警 | 🔴 危险 | 大幅减仓风险资产，转向防御（现金、短期国债） |
| 4 个预警 | 🔴🔴 危机级别 | 最大限度降低风险敞口，对冲尾部风险 |

## 数据获取方式（3 种，按优先级）

### 方式 1：使用 web_fetch 抓取官方数据源（推荐，无需 API key）

**美联储资产负债表**：
- `web_fetch https://www.federalreserve.gov/monetarypolicy/bst_recenttrends.htm`
- `web_fetch https://fred.stlouisfed.org/series/WALCL`（总资产）
- `web_fetch https://fred.stlouisfed.org/series/WGTREASGCA`（TGA）
- `web_fetch https://fred.stlouisfed.org/series/RRPONTSYD`（ON RRP）

**SOFR 利率**：
- `web_fetch https://www.newyorkfed.org/markets/reference-rates/sofr`

**MOVE 指数**：
- `web_fetch https://www.tradingview.com/symbols/MOVE/`

**USDJPY 和国债收益率**：
- `web_fetch https://www.tradingview.com/symbols/USDJPY/`
- `web_fetch https://fred.stlouisfed.org/series/DGS2`（美国 2 年期）

### 方式 2：用户提供数据

用户提供关键指标，例如：
> "美联储总资产 7.8 万亿，TGA 8000 亿，ON RRP 5000 亿，SOFR 5.3%，MOVE 95，USDJPY 148"

### 方式 3：web_search（需要 Brave API key）

如果配置了 API，使用 web_search 搜索最新数据。

## 执行步骤

1. **获取数据**（按优先级：web_fetch → 用户提供 → web_search）
2. 逐一评估每个指标的状态
3. 统计预警数量，得出综合评级
4. 搜索未来数周的关键事件日历
5. 按输出模板生成报告并附上数据来源链接

## 降级模式

如果无法获取实时数据：
- 输出分析框架模板，标注"数据待填充"
- 基于已知信息和定性分析提供洞察
- 明确说明数据时效性和局限性

- 净流动性的三个构成部分需要分别搜索
- SOFR 数据由纽约联储每日发布
- MOVE 指数可从 TradingView 或财经新闻中获取
- 如果某个数据搜索不到最新值，注明最近可获取的数据及其日期
- 所有建议仅供参考，不构成投资建议
