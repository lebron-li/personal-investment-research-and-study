# A 股小盘成长股发现器

发现被市场忽视的小市值高成长 A 股公司。

## 筛选标准

### 市值范围
- 小盘：20-100 亿元
- 中小盘：100-200 亿元

### 成长性指标

| 指标 | 标准 |
|------|------|
| 营收增速 | 近 3 年 CAGR > 20% |
| 利润增速 | 近 3 年 CAGR > 25% |
| ROE | > 15% |
| 毛利率 | 稳中有升 |
| 研发投入 | 占收入 > 5%（科技类） |

### 质量指标

| 指标 | 标准 |
|------|------|
| 经营现金流 | 近 3 年累计为正 |
| 资产负债率 | < 60% |
| 应收账款 | 增速不超过收入 |
| 存货 | 周转率稳定或改善 |

### 特色标签（加分项）

- 专精特新"小巨人"
- 制造业单项冠军
- 进口替代逻辑
- 行业细分龙头

## 工作流程

1. 应用筛选条件
2. 排除风险公司（ST、高质押、高商誉）
3. 深度分析业务模式
4. 评估成长可持续性
5. 给出合理估值区间

## 数据获取方式

**东方财富**：https://data.eastmoney.com/
**工信部**：专精特新名单
**同花顺**：http://data.10jqka.com.cn/

## 重要提醒

- 小盘股波动大，需控制仓位
- 警惕伪成长（靠并购/补贴）
- 关注流动性风险
- 注册制下壳价值下降
- 以上分析仅供参考，不构成投资建议
