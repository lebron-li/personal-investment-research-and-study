# A 股持仓日报自动生成器

## 功能概述

每个交易日自动分析用户持仓，生成两份报告：
- **午间快评** (11:35): 30 秒快读 + 下午操作建议
- **晚间复盘** (15:30): 详细分析 + 明日策略 + 关键位置

紧急情况下单独推送（突破/跌破关键位、成交量异常等）。

## 核心功能

### 1. 资金面分析
| 指标 | 说明 | 数据来源 |
|------|------|---------|
| 北向资金 | 外资净流入/流出 | 东方财富 |
| 主力资金 | 大单净流入额 | 东方财富 |
| 成交量异动 | 量比、换手率、放量/缩量 | 新浪财经 |

### 2. 共振分析
| 维度 | 分析内容 |
|------|---------|
| 板块联动 | 个股 vs 所属行业指数 |
| 大盘共振 | 个股 vs 上证指数/沪深 300 |
| 多周期共振 | 日线 + 周线 + 60 分钟三重确认 |

### 3. 关键位置
- 支撑位/阻力位自动识别
- 前高/前低扫描
- 缺口检测
- 突破/跌破预警（距离<3% 时提醒）

### 4. 事件驱动
- 财报日历（未来 7 天）
- 宏观数据公布日
- 行业政策新闻扫描

### 5. 输出与推送
- 飞书群聊消息推送
- 本地 Markdown 文件保存
- 紧急预警单独推送

## 使用方法

### 手动触发
```
"生成午间持仓日报"
"生成晚间持仓复盘"
```

### 定时任务（自动）
- 交易日 11:35 → 午间快评
- 交易日 15:30 → 晚间复盘
- 紧急情况 → 实时推送

## 配置说明

### 持仓文件
路径：`C:\Users\<USERNAME>\Desktop\The stocks and ETFs I bought.txt`
格式：
```
1、食品饮料 ETF 515170
2、美的集团 000333
...
```

### 报告输出
路径：`C:\Users\<USERNAME>\Desktop\持仓日报\`
命名：`持仓日报_YYYY-MM-DD_午间.md` / `持仓日报_YYYY-MM-DD_晚间.md`

### 飞书推送
- 推送到飞书群聊
- 午间版：简洁摘要
- 晚间版：详细报告链接

## 交易日判断

自动判断是否为 A 股交易日：
- 排除周六、周日
- 排除中国法定节假日
- 参考上交所休市安排

## 紧急预警触发条件

| 条件 | 阈值 |
|------|------|
| 突破关键阻力位 | 涨幅>3% 且突破 |
| 跌破关键支撑位 | 跌幅>3% 且跌破 |
| 成交量异常 | 量比>3 |
| 单日涨跌 | 涨跌幅>7% |

## 注意事项

1. 数据来源：新浪财经、东方财富、巨潮资讯
2. 分析频率：每个交易日两次
3. 报告仅供参考，不构成投资建议
4. 需要配置飞书推送权限

## 依赖项
