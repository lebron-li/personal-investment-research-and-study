# A 股事件驱动机会识别器

识别和分析 A 股公司事件驱动的投资机会。

## 事件类型

### 1. 资产重组/注入

| 维度 | 分析要点 |
|------|---------|
| 资产质量 | 注入资产盈利能力、成长性 |
| 估值 | 注入价格是否合理 |
| 协同效应 | 与现有业务的协同 |
| 审批风险 | 证监会/国资委审批 |

### 2. 国企改革

| 类型 | 机会 |
|------|------|
| 混改 | 引入战投、股权激励 |
| 专业化整合 | 同业资产整合 |
| 分红提升 | 国资委分红率要求 |

### 3. 分拆上市

| 要点 | 说明 |
|------|------|
| 估值重估 | 子公司独立估值可能更高 |
| 聚焦主业 | 母公司更聚焦 |
| 融资渠道 | 子公司独立融资 |

### 4. 回购/增持

| 类型 | 信号强度 |
|------|---------|
| 注销式回购 | 最强 |
| 股权激励回购 | 中等 |
| 大股东增持 | 较强 |
| 响应政策增持 | 较弱 |

### 5. 限售股解禁

| 风险 | 说明 |
|------|------|
| 解禁规模 | 占流通盘比例 |
| 解禁股东 | 创投/财务投资人减持意愿强 |
| 股价位置 | 破发股票解禁压力大 |

## 工作流程

1. 事件识别与分类
2. 历史案例对比
3. 影响量化评估
4. 时间窗口判断
5. 风险提示

## 数据获取方式

**巨潮资讯**：http://www.cninfo.com.cn/
**东方财富**：https://data.eastmoney.com/

## 重要提醒

- 事件驱动需要快速反应
- 警惕"利好出尽"
- 审批风险需充分考虑
- 以上分析仅供参考，不构成投资建议
