# A 股投资适当性报告生成器

生成符合证监会适当性管理框架的投资报告。

## 适当性框架

### 投资者分类

| 类型 | 标准 |
|------|------|
| 保守型（C1） | 保本为主，不能接受损失 |
| 稳健型（C2） | 可接受小幅波动 |
| 平衡型（C3） | 可接受中等风险 |
| 成长型（C4） | 可接受较高风险 |
| 进取型（C5） | 可接受高风险 |

### 产品/服务风险等级

| 等级 | 产品示例 |
|------|---------|
| R1 | 国债、货币基金 |
| R2 | 债券基金、银行理财 |
| R3 | 混合基金、股票基金 |
| R4 | 股票、私募 |
| R5 | 衍生品、高杠杆产品 |

### 匹配原则

- C1 只能买 R1
- C2 可买 R1、R2
- C3 可买 R1、R2、R3
- C4 可买 R1-R4
- C5 可买 R1-R5

## 特殊板块适格条件

### 科创板

| 条件 | 要求 |
|------|------|
| 资产 | 前 20 交易日日均≥50 万 |
| 经验 | 证券交易≥24 个月 |
| 风险等级 | C4 及以上 |

### 北交所

| 条件 | 要求 |
|------|------|
| 资产 | 前 20 交易日日均≥50 万 |
| 经验 | 证券交易≥24 个月 |
| 风险等级 | C4 及以上 |

### 创业板

| 条件 | 要求 |
|------|------|
| 资产 | 前 20 交易日日均≥10 万 |
| 经验 | 证券交易≥24 个月 |

## 报告内容

1. 投资者基本信息
2. 风险测评结果
3. 拟投资产品/服务
4. 匹配度评估
5. 风险揭示
6. 投资者确认

## 重要提醒

- 适当性管理是监管要求
- 不得向不匹配投资者推荐高风险产品
- 需投资者签字确认
- 定期重新评估（至少每 2 年）
