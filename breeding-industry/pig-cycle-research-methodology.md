## 一、核心规律：股价领先猪价 6-12 个月

这是整个框架的基石。每次都是在行业深度亏损、散户最绝望时股价先见底。

| 周期 | 股价见底 | 猪价见底 | 领先时间 |
|------|---------|---------|---------|
| 2014 年 | 2014 年初 | 2014 年中 | ~6 个月 |
| 2018 年非瘟 | 2018/10 | 2019/02 | ~4 个月 |
| 2022-2023 年 | 2023/10（31 元） | 2024/05 | ~7 个月 |

**推论**：等所有人都看到猪价涨了，股价早已脱离底部。底部建仓的核心不是判断"猪价到底没有"，而是判断"绝望程度够不够"。

## 二、五大"All In"信号体系

> 这是判断猪周期底部最核心的量化框架。信号越多共振，底部确定性越高。

### 信号①：淘汰母猪价格 < 3 元/斤 ⭐⭐⭐⭐⭐

**为什么是最硬指标**：养殖户淘汰母猪 = 放弃未来一年的生产能力，这是绝望的终极信号。历史上每次猪周期大底，这个信号从未缺席。

**数据来源**：搜 "淘汰母猪价格 元/斤" → 养猪网 / 猪易网 / Mysteel

| 状态 | 淘汰母猪价格 | 含义 |
|------|:---:|------|
| ❌ 未触发 | > 4 元/斤 | 养殖户还在扛 |
| ⚠️ 接近 | 3-4 元/斤 | 恐慌开始蔓延 |
| ✅ 触发 | < 3 元/斤 | 历史级绝望 — 抄底最强信号 |

### 信号②：能繁母猪连续 3 月环比降幅 > 1% ⭐⭐⭐⭐⭐

**为什么是先行指标**：能繁去化 → 10 个月后供给收缩 → 猪价拐点。环比降幅的**加速度**比绝对值更重要。

**数据来源**：农业农村部月度发布（每月中旬公布上月数据）/ 搜 "能繁母猪存栏环比"

**判断逻辑**：
```
连续 3 个月环比降幅 > 1% → 去化趋势确认
单月降幅 > 2%           → 加速去化，重大信号
能繁降至 3650 万以下    → 考虑到 PSY 提升，实质供给收缩
```

### 信号③：头均市值 < 3500 元 ⭐⭐⭐⭐

**计算方式**：
```
头均市值 = 总市值 / 年出栏量
牧原：~2012 亿 / ~7800 万头 ≈ 2580 元/头  (6/15)
温氏：~917 亿 / ~3500 万头 ≈ 2620 元/头
新希望：~365 亿 / ~1800 万头 ≈ 2028 元/头
```

| 区间 | 含义 |
|------|------|
| > 5000 元 | 估值偏高 |
| 3500-5000 | 正常区间 |
| 2500-3500 | 低估区 ✅ 触发 |
| < 2500 | 深度极值（过去 5 年仅 2 次） |

### 信号④：行业现金流危机（暴雷/倒闭） ⭐⭐⭐⭐

**为什么要等暴雷**：猪周期底部的历史规律——高杠杆的先倒下，活下来的才是王者。暴雷是散户最恐慌的时刻，也是底部确认的重要信号。

**观察清单**：
- 上市公司经营现金流转负（牧原 Q1 已触发：-9.2 亿）
- 资产负债率 > 70% 的企业出现债务危机
- 正邦/天邦级别的暴雷事件重现

### 信号⑤：政策强力干预 ⭐⭐⭐

**已触发事件**（2026/05）：
- 能繁母猪目标 4100 → 3750 万头（硬性约束头部出栏量）
- 金融端收紧："原则上不支持生猪养殖场扩大生产"
- 冻猪肉收储部署

**注意**：政策底 ≠ 市场底，历史上政策底后通常还要磨 3-6 个月。

## 三、本轮周期特殊风险

这三个因素让本轮周期比以往更复杂：

### 1. PSY 效率提升 43% 抵消能繁去化
```
2018 年：PSY 从 17 提升至当前 24.34（+43%）
3904 万头能繁 × 24.34 PSY ≈ 历史 4500 万头产能
能繁可能要降到 3600 万以下才能真正收缩供给
```

### 2. 头部企业"熬死对手"策略拉长底部
- 牧原账面资金 166 亿 + 银行授信
- 温氏负债率 58%，财务最安全
- 去化以中小散户为主，速度比历史慢

### 3. 牧原 H 股摊薄
- 总股本 54.63 → 57.73 亿股
- EPS 被摊薄约 5.7%
- 每股弹性比以前弱
