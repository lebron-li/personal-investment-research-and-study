```
├── banks-stocks/          # 银行股深度分析（招商银行、兴业银行等）
```

## 技能集

`skills/` 目录包含 20+ 个金融分析技能，覆盖：

- **估值分析**：DCF/DDM/PB-ROE、科技股估值、低估值筛选、财报深度分析
- **组合管理**：组合健康诊断、风险收益优化、长期建仓助手（v2.3：TSR综合回报/AI叙事风险/跨市场比价）
- **策略量化**：多因子筛选、高股息策略、行业轮动
- **事件驱动**：资产重组/国企改革、董监高增减持、情绪偏差、小盘成长
- **技术分析**：日线/周线技术面、持仓日报、午间快报
- **宏观与ESG**：宏观流动性监控、宏观经济数据查询、ESG筛选、投资适当性

详见 [`skills/README.md`](skills/README.md)
