# 个人投资研究知识库

宏观经济、行业研究、估值模型、量化策略的个人知识库。

## 目录结构

```
├── innovative-pharma/     # 创新药与医疗板块研究
├── kondratieff-cycle/     # 康波周期研究
├── precious-metals/       # 贵金属（黄金）趋势分析
├── real-estate/           # 房地产估值模型
└── skills/                # 金融分析技能集（20个）→ 详见 skills/README.md
```

## 技能集

`skills/` 目录包含 20 个金融分析技能，覆盖：

- **估值分析**：DCF/DDM/PB-ROE 方法论、科技股估值、低估值筛选、财务报表深度分析
- **组合管理**：组合健康诊断、风险收益优化、长期建仓助手
- **策略量化**：多因子筛选、高股息策略、行业轮动
- **事件驱动**：资产重组/国企改革、董监高增减持、情绪偏差、小盘成长
- **技术分析**：日线/周线技术面、持仓日报、午间快报
- **宏观与ESG**：全球流动性监控、ESG 筛选、投资适当性

详见 [`skills/README.md`](skills/README.md)
